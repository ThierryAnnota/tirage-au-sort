Notice d'utilisation

Respect du RGPD

✅ Cet outil a été uniquement conçu en front-end. Il n'y a pas de back-end : aucune donnée n'est envoyée sur un serveur ; tout ce que vous entrez dans l'outil (les noms pour le tirage au sort) reste uniquement sur votre appareil. Rien n'est stocké en ligne.

✅ Aucune inscription, aucun suivi : vous n'avez pas besoin de créer un compte, et aucune donnée personnelle n'est enregistrée.

✅ Pas de cookies, pas de pub : Ce site est 100% gratuit et respecte votre vie privée.

🌐 Connexion Internet nécessaire : uniquement pour accéder au site, mais ensuite, tout fonctionne localement sur votre appareil.

Le code est librement consultable sur github (lien en bas de page en cliquant sur le logo).

💡 Comment ça marche ?

Entrez simplement les noms un par un ou les séparant par des virgules.

Petite astuce : si vous utilisez souvent les mêmes noms (de personnes ou d'objets) pour le tirage au sort, il peut être utile d'enregistrer cette liste (en séparant les noms par des virgules) dans un fichier ".txt".

NB : "Effacer" permet de tout effacer (ce qui est plutôt logique), tel que l'on pourrait le faire en raffraichissant la page ("F5", "Cmd/Ctrl + R").

"Recommencer" permet de relancer un tirage avec les candidats présents sur la page (qui ne seront pas forcément dans le même ordre)

Lorsque le tirage au sort est terminé, tant qu'il reste des candidats, un bouton apparaît permettant de remettre une pièce dans la machine.

Il est possible de corriger l'orthographe d'un.e candidat.e (en cliquant sur son nom). La fonction disparaît lorsqu'il ou elle est sélectionné.e.
